# CJ-6

## Description
 A drone designed in Solid Edge and programed in C++ able to follow a line to solve a maze, using ultrasonic and infrared sensors.
 - Language: C++.
 - Programs: Solid Edge / Arduino IDE.
 - Role: 3D Designer
